Dark color theme for VSCode, inspired by the [2017 Windows Console colors](https://devblogs.microsoft.com/commandline/updating-the-windows-console-colors/), now called "Campbell".

| Hex | Color Name | Usage |
|---:|---|---|
| **Campbell Palette** | | |
| #0c0c0c | Campbell Black | Secondary backgrounds, borders |
| #0037da | Electric Blue | Primary buttons, keywords (storage) |
| #13a10e | Terminal Green | Status indicators, remote items |
| #3a96dd | Sky Blue | Links, type references, CSS properties |
| #c50f1f | Signal Red | Debugging background |
| #881798 | Magenta Purple | Keywords, preprocessor directives |
| #c19c00 | Mustard Gold | Escape sequences, CSS classes, decorators |
| #cccccc | Light Gray | Main foreground, normal text |
| #767676 | Neutral Gray | Comments, inactive elements, borders |
| #3b78ff | Bright Blue | Focus borders, active elements, constants |
| #16c60c | Bright Green | Numbers, added lines, success states |
| #61d6d6 | Aqua Teal | Classes, enums, interfaces, JSON keys |
| #e74856 | Coral Red | Errors, deleted lines, regex groups |
| #b4009e | Deep Fuchsia | Control flow keywords, language variables |
| #f9f1a5 | Pale Butter | Functions, CSS selectors |
| #f2f2f2 | Off White | Active tab text, status bar text |
| **Additional Colors** | | |
| #000000 | True Black | Main backgrounds (editor, sidebar, panels) |
| #d4956c | Warm Clay | Strings (muted warm tone) |
| #c96a6a | Dark Muted Red | Regular expressions |
| #7db8b8 | Blueish Teal | Interfaces (distinct from classes) |
| #a8d7f7 | Light Sky | Object properties, labels |
| #c9c9c9 | Pale Gray | Function parameters |
